# Kings Bakes Website – Product Requirement Document (PRD)

## 1. Project Overview

**Project Name:** Kings Bakes Bakery Website
**Developed By:** (Your Name)
**Purpose:** To create a professional online presence and ordering support website for a local bakery.

### Business Details

* **Shop Name:** Kings Bakes
* **Address:** Sathanur Main Rd, Sundar Nagar, Tiruchirappalli, Tamil Nadu – 620021
* **Phone:** 076392 77171
* **Opening Hours:** 9:00 AM – 10:00 PM
* **Current Online Presence:**

  * WhatsApp ordering (wa.me link)
  * Zomato listing
  * Instagram page
  * Facebook page
  * wheree.com listing

The shop currently does not have a dedicated professional website.

---

## 2. Problem Statement

The bakery depends only on social media and food listing platforms. Because of this:

* Customers cannot easily view full menu
* No direct ordering platform
* Difficult to find business details in one place
* Limited brand identity online
* Depends on third-party platforms

---

## 3. Project Objective

The objective of this project is to develop a responsive website that:

* Shows bakery products and prices
* Provides shop information
* Enables customers to order via WhatsApp
* Displays location using Google Maps
* Improves online visibility and branding

---

## 4. Target Users

1. Local residents of Tiruchirappalli
2. College students
3. Birthday cake customers
4. Event and party customers

---

## 5. Website Features

### 5.1 Home Page

* Shop banner image
* Short introduction
* Featured products
* Opening hours
* WhatsApp order button

### 5.2 About Page

* Bakery description
* Address details
* Embedded Google Map location

### 5.3 Menu Page

Product categories:

* Cakes
* Pastries
* Puffs
* Snacks
* Beverages

Each product will contain:

* Product image
* Name
* Price
* Description

### 5.4 Order Feature

Customers can click **Order Now** button which opens WhatsApp with a pre-filled message:

"Hi Kings Bakes, I want to place an order."

### 5.5 Contact Page

* Phone number
* Address
* Google Map
* Contact form (Name, Phone, Message)
